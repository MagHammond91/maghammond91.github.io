Case Study

Converting a PDF Income Statement into a Structured and Presentable Excel File

The client provided a PDF version of a company's income statement, which needed to be accurately converted into a well-structured and formatted Excel file. The final deliverable had to maintain the integrity of financial data, ensuring that all rows and columns aligned properly for easy review, analysis, and reporting.

Deliverables

  • A structured Excel file replicating the original income statement format.
  • Accurate financial data correctly aligned in respective rows and columns.
  • Error-free calculations ensuring that net income and other financial metrics match the source document.
  • Proper formatting of figures.

Tools Used

  • Microsoft Excel
  • PDF2Go-PDF Editor & Converter
  • Google Drive

Process & Approach

I uploaded the PDF income statement to Google Drive. Opened the file using PDF2Go - PDF Editor and Converter to convert it into an Excel file (.xlsx). Downloaded the converted Excel file for further processing. Opened the converted Excel file and reviewed the layout, alignment, and accuracy of extracted financial data. Adjusted misplaced columns, removed unnecessary text, and corrected merged cells to ensure all financial figures were properly aligned. Deleted blank rows and fixed broken numerical values to match the original statement format.

I then applied appropriate currency formatting for revenue, expenses, and net income. Used Excel formulas (such as SUM and IF) to ensure accurate subtotals and total calculations. Structured the financial statement headings, categories, and sections for clarity and readability. Cross-checked the Excel file with the original PDF to ensure all figures matched accurately. Verified that all calculations were correct and that formatting followed proper financial reporting standards. Lastly, made final adjustments to layout, alignment, and presentation to ensure a clean and professional result.
